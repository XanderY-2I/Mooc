## 总体概览

| 维度 | 原始 Demo | 扩展后 |
|------|----------|--------|
| 微服务数量 | **13** 个（含3个基础模块） | **18** 个 |
| 数据库表 | ~40 张 | ~60+ 张（新增 20+ 张） |
| 前端页面 | 基础功能页面 | 大量新增页面（私信、群聊、AI对话、积分商城、数据大屏等） |
| 中间件 | MySQL、Redis、RabbitMQ、ES、Nacos、Seata | 基础上 + MinIO、Sentinel、InfluxDB、Qdrant、Prometheus+Grafana、XXL-JOB |
| 代码提交数 | ~40 次（虎哥） | ~100+ 次（付申祺） |

| 序号 | 模块 | 说明 |

## 基础设施模块

### 1. `tj-common` — 公共模块

| 原始 Demo | MyBatis-Plus 配置 |
|           | 基础工具类（JSON、日期等） |
|           | 统一返回类 |
|           | 全局异常处理 |

|           | XXL-JOB 配置类 |

### 2. `tj-api` — Feign 接口定义模块

| 原始 Demo | 认证、用户、课程、搜索等基础 Feign Client |
|           | 课程目录、分类、科目等辅助 Client |
|           | 基础 DTO 定义 |
| **扩展新增** | 学员专用 Feign 接口（注册、修改密码、解绑手机号），配备降级容错 |
|           | 通用点赞状态查询接口 |
|           | 优惠券退还等促销服务调用接口 |
|           | 收藏/评价/积分/私信/群聊等大量新 DTO 和 VO |

### 3. `tj-gateway` — 网关

| 原始 Demo | 12 条路由规则（覆盖所有原始微服务） |
|           | JWT 鉴权过滤器 |
|           | Host 头传递 |
| **扩展新增** | WebSocket 路由 — 支持 ws:// 协议转发到消息微服务（在线群聊） |
|           | SSE 流式路由 — 支持 AI 对话的 Server-Sent Events |
|           | InfluxDB 日志埋点过滤器 — 自定义过滤器，捕获每个请求的状态码/消息/耗时并存入 InfluxDB |
|           | 网关层用户信息提取（城市、性别、姓名）传递到下游 |
|           | 课程 ID 校验拦截 — 拦截恶意流量请求不存在的课程 |
